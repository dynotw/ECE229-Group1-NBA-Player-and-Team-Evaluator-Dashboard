Attributes_Distribution.distribution(season, graph)[source]¶

Interface for creating interactive plots.

Parameters

  • season (int) – year of desired season (ex. 2018)

  • graph (int) – selects which graph is to be created (1 = Distribution, 2 = Bi-Distribution, 3 = Contour, 4 = KDE)
