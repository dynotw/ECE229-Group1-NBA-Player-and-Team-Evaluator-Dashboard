BestFeatures module¶

BestFeatures.best_player_features(season=2019, label='PTS', top=6)[source]¶

This function returns the plot for the top k attributes that contribute the most to the chosen y value using linear reggression. This is based on player data

Parameters

  • season (int) – the season for which you want to plot the data for

  • label (string) – the y-value (main attribute) that you want to find the top k atrributes for

  • top (int) – the number of k attributes

Returns

Bar plot of the top k choosen atrributes and their weights for the players data set
