Set Up and Usage Guide¶

Required Packages and Dependencies¶

  • numpy

  • pandas

  • sklearn

  • logging

  • requests

  • json

  • math

  • matplotlib

  • seaborn

  • plotly express

  • plotly graph_objs

  • cufflinks

  • ipywidgets

For installing these packages, you can use either pip or conda to install packages. For example,

pip3 install numpy

or if you are using conda,

conda install numpy

Usage¶

  1. Download the source repository from here.

  2. Open the Main.ipynb Juptyer Notebook to see all of the visualizations.

jupyter notebook Main

  1. To test the codebase run this command in the repo directory.

python3 -m pytest --con-report -term --con=/pyTest
