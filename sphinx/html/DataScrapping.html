DataScrapping module¶

DataScrapping.get_player_data(sea, cat)[source]¶

Creates a Pandas Dataframe of the stats for a given season and category. For example, sea = ‘2019-2020’ and cat = ‘AST’, will get the amount of assists for players in the 2019-2020 season sorted by mosts assists.

Parameters

  • sea (string) – The NBA Regular Season for which you want to retrive data

  • cat (string) – The Category for which you want the Pandas data frame to be order in

Returns

A Pandas DataFrame of the stats in respect to the above mentioned criteria

Return type

pd.DataFrame

DataScrapping.shot_ranges(sea)[source]¶

Given a season (ex. 2019-2020), this will return a DataFrame of the players shot statistics from different ranges on the basketball court.

Parameters

sea (string) – The NBA Regular Season for which you want to retrieve data. (ex. ‘2019-2020’)

Returns

A Pandas DataFrame of the shot statistics from different distances.

Return type

pd.DataFrame
